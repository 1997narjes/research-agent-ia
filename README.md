# 🔍 Agent de Recherche Web IA

Agent IA qui recherche sur internet, lit les articles et génère des rapports.

## Technologies
- Python
- Groq (Llama 3.3)
- DuckDuckGo Search
- Newspaper3k (scraping)

## Fonctionnalités
- Recherche web en temps réel
- Lecture et scraping d'articles
- Résumé intelligent par IA
- Synthèse finale structurée
- Sauvegarde rapport Markdown

## Installation
pip install groq ddgs newspaper3k lxml_html_clean python-dotenv

## Configuration
Crée un fichier .env :
GROQ_API_KEY=ta_clé_ici

## Utilisation
python research_agent.py
